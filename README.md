## # Mastralis :
Mastralis is a DotNet simple solo trainer with a few features such as :
* Set your own attack speed
* Set your own casting speed
* Set your own movement speed
* Inifinite HP
* Infinite Mana
* Infinite Stamina
* Infinite Karma
* Infinite Magic DEF
* Infinite Phy DEF
* Infinite Posture RES
* One Hit One Kill

**More features will be added.**
